# Sales Data Analysis Dashboard

This project demonstrates data analysis skills using Python's pandas library and visualization tools.

## Overview

Analyzes a sample retail sales dataset to extract insights about:
- Sales trends over time
- Top performing products
- Regional performance analysis
- Customer segmentation

## Features

- Data cleaning and preprocessing
- Statistical analysis and aggregations
- Multiple visualization types (line charts, bar charts, heatmaps)
- Export results to CSV and charts to PNG

## Requirements

See `requirements.txt` for dependencies.

## Usage

```bash
python sales_analysis.py
```

The script will:
1. Load and clean the sample data
2. Perform analysis
3. Generate visualizations in the `output/` directory
4. Print summary statistics to console

## Sample Output

- `output/sales_trend.png` - Monthly sales trend
- `output/top_products.png` - Top 10 products by revenue
- `output/regional_analysis.png` - Sales by region
- `output/summary_report.csv` - Detailed summary statistics
